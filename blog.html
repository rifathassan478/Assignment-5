Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM: 1. `getElementById()`: Selects by ID. 2. `getElementsByClassName()`: Selects by class name. 3. `getElementsByTagName()`: Selects by tag name. 4. `querySelector()`: Selects the first maching CSS selector. 5. `querySelectorAll()`: Selects all matching CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

`innerHTML` returns the HTML content of an element, `innerText` returns the visible text content, and `textContent` returns all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to manage events for multiple child elements, improving performance and simplifying code.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and propagates up to the root, allowing parent elements to handle events triggered by their children.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements using JavaScript, you can use the following methods: 1. `document.createElement(tagName)`: Creates a new element with the specified tag name. 2. `parentElement.appendChild(newElement)`: Adds the new element as the last child of the parent element. 3. `parentElement.insertBefore(newElement, referenceElement)`: Inserts the new element before the reference element within the parent element. 4. `element.remove()`: Removes the specified element from the DOM. 5. `parentElement.removeChild(childElement)`: Removes the specified child element from the parent element.
